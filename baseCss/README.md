####说明：
```
1、使用前请先大概阅读一下base.css的内容，注释比较清楚
2、如果是小程序项目，请去掉 '格式化 web H5 基础样式'模块
3、其中有关于颜色的控制 --> 颜色模块 --> ...  （利于维护，和修改）
4、清除浮动，左右浮动，边框阴影等 在 --> 小功能样式模块
5、关于弹性盒的只有 ak-flexC/flexS/flexE/flexB/flex/flex-columnC （推荐在 手机端 使用）

```
